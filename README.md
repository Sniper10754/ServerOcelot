
# ServerOcelot

A brief description of what this project does and who it's for


## Run Locally

Clone the project

```bash
  git clone https://github.com/Sniper10754/ServerOcelot
```

In project directory run: 
```bash
  make run 
```
Or if you don't have make installed:
```bash
  python3 main.py
```
